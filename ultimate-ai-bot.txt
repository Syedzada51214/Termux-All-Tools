# Ultimate AI Telegram Bot 🤖

A modular Telegram bot with AI capabilities, built for extensibility and ease of use.

## Features ✨

- **Plugin Architecture** - Add new features easily
- **AI Integration** - OpenAI GPT for smart responses
- **Multiple Services** - Weather, crypto, reminders
- **Admin Controls** - Secure access to privileged commands

## Quick Start 🚀

1. Clone the repo:
   ```bash
   git clone https://github.com/syedzada51214/ultimate-ai-bot.git
   cd ultimate-ai-bot
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure environment:
   ```bash
   cp .env.example .env
   ```
   Edit `.env` with your API keys.

4. Run the bot:
   ```bash
   python bot.py
   ```

## Commands 📝

| Command | Description |
|---------|-------------|
| `/start` | Start the bot |
| `/help` | Show available commands |
| `/weather <location>` | Get weather info |
| `/remind <task> in <time>` | Set a reminder |
| `/crypto <coin>` | Check crypto prices |
